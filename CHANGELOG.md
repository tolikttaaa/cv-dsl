# Changelog

All notable changes to this project are documented here. Releases follow
[Semantic Versioning](https://semver.org/) and are automated by Release Please.

## [0.1.0] - 2026-07-16

### Features

- type-safe Kotlin DSL and immutable CV model;
- static web and LuaLaTeX renderers from one source;
- bundled browser assets, LaTeX class and local fonts;
- Gradle generation, PDF, site assembly and preview tasks;
- standalone example, contributor documentation and automated releases.
